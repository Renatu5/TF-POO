Esse ReadMe serve para avisar sobre um ponto importante da aplicação, que são as fontes.
Fiz o reenvio do exercício pois percebi que as fontes personalizadas que o programa usa
não estavam na pasta do código, onde agora estão. No entanto, infelizmente, não consegui fazer com que o programa
"forçasse" a utilização das fontes personalizadas, caso o computador não as tenha instaladas. Tentei utilizar algumas
dicas e métodos de alguns fóruns, mas não tive sucesso ao compilar dessa forma.
Decidi, então, enviar com os arquivos das duas fontes utilizadas, para que, se quiser ter a experiência visual
completa, possa baixar a fonte antes de rodar o programa e ver como ele foi projetado originalmente.
Acabei percebendo esse ponto apenas depois do primeiro envio, pois foi quando loguei em um computador da universidade
que, diferente do meu computador, não tinha as fontes instaladas.

Desde já, obrigado
Att, Caio.
